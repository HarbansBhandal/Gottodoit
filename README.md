# MEAN ToDos

MEAN ToDos is a full MEAN stack To-Do list web app that allows users to save their to-dos, along with past, current, or future due dates. The to-dos can be created, marked completed or reverted. 

On the front end, AngularJS directives, controllers, routes, and a service exist to display, edit, create, and delete within the to-do list application. Also used are Node, Express server, MongoDB with Mongoose ODM, bluebird NPM for promises, morgan NPM for development logging.
